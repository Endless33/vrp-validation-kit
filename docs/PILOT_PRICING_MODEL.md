Pilot Pricing Model

Purpose

This document describes possible pilot engagement structures.

Pricing is intentionally not fixed.

Pilot scope varies significantly depending on environment complexity and validation objectives.

---

Evaluation Factors

Examples:

- Duration
- Environment complexity
- Validation scope
- Required support
- Evidence requirements

---

Engagement Types

Technical Evaluation

Focused validation of selected scenarios.

Examples:

- Replay validation
- Recovery validation
- Authority validation

---

Extended Evaluation

Broader validation across multiple environments and failure classes.

---

Integration Assessment

Evaluation of how VRP concepts may interact with an existing architecture.

---

Notes

Pilot engagements should be scoped before pricing discussions.

Validation objectives should be clearly defined before estimating effort.
